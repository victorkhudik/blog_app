# Blog App

Простой блог на чистом PHP с использованием Smarty и MySQL.

## Технологический стек

- PHP 8.1+
- Smarty 5.0
- MySQL 8.0
- Docker & Docker Compose
- SCSS

## Структура данных

### Категория
- Название
- Описание

### Статья
- Изображение
- Название
- Описание
- Текст
- Категория (одна или несколько)
- Количество просмотров

## Функционал

### Главная страница
- Вывод категорий с 3 последними постами
- Кнопка "Все статьи" для каждой категории

### Страница категории
- Название и описание категории
- Список статей
- Сортировка по просмотрам и дате
- Пагинация

### Страница статьи
- Полная информация о статье
- Блок из 3 похожих статей

## Установка и запуск

### Через Docker

```bash
# 1. Клонировать репозиторий
git@github.com:victorkhudik/blog_app.git
cd blog_app

# 2. Запустить Docker
docker compose up -d

# 3. Установить зависимости
docker compose exec app composer install

# 4. Создаем базу данных и таблицы
docker compose exec app php setup/setup.php

# 5. Создаем тестовые данные
docker compose exec app php setup/seeder.php

# 6. Открыть в браузере
http://localhost:8080
